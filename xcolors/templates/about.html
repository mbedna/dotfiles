About

Xcolors.net is a color theme directory for terminals that support color configuration through X resources. It provides a visual presentation of the themes and will hopefully grow over time with help from the community.

The source code is hosted on github. Head over there to see how this works, contribute or report any issues. Also see this if you want to contribute your own color themes or themes you collected over time.
